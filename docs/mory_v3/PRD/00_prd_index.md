## 3. 产品统一定义

`Mory` 是一个面向长期私人生活材料的记忆操作系统。

它不是：

- 通用笔记
- 任务管理器
- 社交内容平台
- 聊天式 AI 外壳

它提供的是：

- 捕获
- 组织
- 连接
- 回看
- 理解

## 4. 使用原则

1. 先定义对象，再定义页面。
2. 先定义记忆层级，再定义单点功能。
3. 一切需求都必须能落回统一记忆本体。
