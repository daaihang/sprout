## 3. v7 Product Promise

Mory should help the user:

- remember important people and relationships,
- understand emotional patterns over time,
- revisit past decisions with evidence,
- discover repeated themes in life fragments,
- capture context without cognitive load,
- correct the system when it is wrong.

## 4. Acceptance Criteria

v7 is not complete until:

- new AI analysis can use bounded historical evidence,
- the user can correct identity, mood, and graph mistakes,
- corrections affect future analysis,
- self profile and person profiles are editable and evidence-backed,
- background jobs can prepare useful questions/notifications without requiring a foreground Home refresh,
- notification and AI behavior remains privacy-gated and explainable,
- system context sources create normal drafts and evidence, not hidden background records,
- notifications are policy-gated, explainable, and interaction-aware.
